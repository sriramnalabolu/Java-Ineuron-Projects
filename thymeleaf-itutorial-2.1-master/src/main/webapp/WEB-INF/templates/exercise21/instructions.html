This exercise shows the syntax for the template URI variables as described in Link section of the tutorial (search "variable templates").

This syntax is handy for the construction of REST-style URLs.

  1. Two beans, productId and productName have been set into the context.
  2. Use the path variable syntax to build a URL containing this parameters in the form of: http://itutorial.thymeleaf.org/buy/273/Red-carpet.html
  3. You can check the proposed solution by pressing the Show solution button () above the code editor.
